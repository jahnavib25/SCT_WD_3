# Tic-Tac-Toe Web Application

This project was developed as part of the **SkillCraft Technology Web Development Internship**.

## 📖 Project Overview

A modern and interactive Tic-Tac-Toe Web Application built using **HTML5, CSS3, and Vanilla JavaScript**. The application features an attractive user interface, multiple game modes, intelligent AI, animations, sound effects, and a fully responsive design.

## ✨ Features

- 🎮 Player vs Player Mode
- 🤖 Player vs Computer Mode
- 🧠 Three AI Difficulty Levels
  - Easy
  - Medium
  - Hard (Minimax Algorithm)
- 🏆 Win and Draw Detection
- 🔄 Restart and New Game Options
- ↩️ Undo Move
- 🌙 Dark & Light Theme
- 🔊 Sound Effects
- 📊 Live Scoreboard
- 📜 Match History
- 📈 Statistics Dashboard
- 🎉 Confetti Animation on Victory
- 💾 Local Storage Support
- 📱 Fully Responsive Design
- ♿ Keyboard Accessibility

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)
